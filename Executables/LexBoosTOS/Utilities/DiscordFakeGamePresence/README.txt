Discord Fake Game Presence - Portable Version
============================================

This is a portable version of Discord Fake Game Presence.

To use:
1. Extract this ZIP file to any folder
2. Run DiscordFakeGamePresence.exe

The application doesn't require installation and won't
leave traces in the system registry.

Note: .NET 8.0 Desktop Runtime is required to run this application.
If not installed, download from: https://dotnet.microsoft.com/download
