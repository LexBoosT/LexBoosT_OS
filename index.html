Ce que fait LexBoosT OS™

7 catégories d'optimisations pour des performances radicales.

💻

Optimisations Système

  • MMCSS recalibré, TSX activé, ThreadDpcEnable
  • Mitigations Spectre/Meltdown désactivées (+5-15% CPU)
  • Core Parking contrôlable, priorités IFEO par processus
  • DisablePagingExecutive, Memory Compression désactivée
  • BCD tweaks, NTFS optimisé, ServicesPipeTimeout=30s
🌐

Optimisations Réseau

  • Nagle désactivé, TcpTimedWaitDelay=30s
  • MaxUserPort=65534, MaxFreeTcbs=65536
  • DNS-over-HTTPS natif, Cache TTL 24h
  • 29 apps VoIP priorisées (Discord, Teams, Zoom, Skype)
  • EEE désactivé, FlowControl off, RSS activé
🎮

Gaming & GPU

  • MMCSS Games Task: GPU Priority=8, SFIO Priority=High
  • HAGS activé (Hardware-Accelerated GPU Scheduling)
  • Game DVR complètement désactivé (HKLM+HKCU)
  • DWM: MaxQueuedPresentBuffers=2, FrameLatency=1
  • MPO/DWM overlays nettoyés (anti-flickering)
🔒

Privacy Radicale

  • AllowTelemetry=0 sur 5 chemins
  • IA bloquée: generativeAI=Deny, Copilot supprimé
  • Cortana, Advertising ID, Cloud Search désactivés
  • OneDrive KFM bloqué, PerfTrack désactivé
  • Edge telemetry, WER, Feedback Notifications supprimés
🧹

Débloat Agressif

  • 100+ packages AppX supprimés
  • Copilot, Teams, Xbox GameBar, OneDrive, Cortana
  • Bing*, Zune*, Office Hub, Skype, Spotify, LinkedIn
  • Microsoft Edge supprimable, OneDrive désinstallable
  • WinSxS nettoyé, DISM features superflues désactivées
🛡️

Fiabilité & Stabilité

  • Valeurs conservatrices, pas de DPC < 50ms
  • Pas de Large Pages forcées, SEHOP conservé
  • Bugs corrigés: DisableExceptionChainValidation, LargePageMinimum
  • Compatibilité: SysMain=Manual, ClipSVC=Auto (Win+V)
  • TokenBroker=Auto (MS Store), lsass.exe PerfOptions

UX & Personnalisation

  • Thème dark personnalisé, wallpapers, avatar

Installation

Déployez LexBoosT OS™ en quelques minutes.

1

Prérequis

Windows 11 propre (builds 22000-28020), pilotes à jour, apps Store à jour, Widgets connectés au compte Microsoft. Créez un point de restauration système. Redémarrez.

2

Lancez le Playbook

Téléchargez et exécutez le playbook LexBoosT OS™ depuis le repo GitHub.

3

Suivez les options

Choisissez votre navigateur, les packages à supprimer, le thème et les personnalisations.

4

Redémarrez

Une fois l'installation terminée, redémarrez pour appliquer tous les changements.

💡

Compatibilité matérielle : AMD Ryzen / Intel Core 8ᵉ gén+, 8 GB RAM minimum (16 GB recommandé), SSD/NVMe recommandé.
